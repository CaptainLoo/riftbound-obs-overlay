Stream Deck — ready to use

          

            Install the plugin once, then download a profile generated from your current decks.
            Every card becomes a button with its art — no URLs to copy. Re-download after deck changes.
          


          
    
            
  1. 
              Install the plugin first — Windows: double-click
              Install Stream Deck plugin.bat
              in the app folder. Then quit and reopen Stream Deck completely.
            
    2. 
            
  2. 
              Verify the plugin — drag any action onto a key. You must see a
              Riftbound OBS category (Show Card, Matchup…). If missing, fix step 1 before importing.
            
    3. 
            
  3. 
              Download the profile — pick the device that matches your hardware
              (15-key classic = Standard, not XL). Click Download profile.
            
    4. 
            
  4. 
              Import — double-click the file, click Install. Then in Stream Deck:
              select your device on the left, open the profile dropdown at the top,
              choose Riftbound OBS. Or: Settings → Profiles → look for Riftbound OBS.
            
    5. 
          


          

            Yellow triangle on a key? The Riftbound app is not running — launch
            Start Riftbound.bat and keep that window open while streaming.
            Reinstall the plugin after updating the app zip.
